For I, the website that should have said a little about me, I learned AngularJS,^^ a front-end web framework. The powerful and innovative MVW JavaScript framework for writing applications by Google getting a Single Page Application. AngularJS lets extend HTML’s syntax, automatically synchronizes data from UI (view) with JavaScript objects (model) through 2-way data binding and teaches the browser how to do dependency injection and inversion of control.
MVW (Model-View-Whatever)^ is an evolution of the programming model MVC. The Model-View-Controller (MVC^) architectural pattern separates an application into three main components: the model (M), the view (V), and the controller (C) providing loose coupling between different aspects of the application, input logic, business logic, and User Interface (UI) logic. It enables to focus on one aspect of the implementation at a time as separation of concerns. Over time and thanks to many refactorings and api improvements, AngularJS is now closer to MVVM, the $scope object could be considered the ViewModel that is being decorated by a function (Controller). This framework flexibility has been named as whatever (W), whatever works for you.

For the interface, the component library is Bootstrap,^ the most popular HTML, CSS and JS library in the world.
I is in Italian and English, the two languages management has required an extra effort for the routing and the definition of the controllers.
Short snippets of code in Javascript implemented by me handle these functions:

  • create the table of contents
  • creates a progressive number, which is a hypertext link that sends to the quotation of the source, contained in the notes
  • creates a hyperlink, symbolized with ^, from the quote to the text
  • list links' domain names
  • list photos' domain names
  • with addthis^ add social and email links

Hyperlinks that do not address the site or Wikipedia have an additional symbol. The site has a small library of 16x16 graphic symbols of the sites to which you refer; if the symbol is existing in my library is added in front of the hyperlink, otherwise the external connection is highlighted with a standard symbol after link. The library was built with search icons by Google Search with the site name and image of the size 16x16. If the brand icon of the site is available as a resource on the Internet is added to the library.
The link is highlighted when the mouse is over.
